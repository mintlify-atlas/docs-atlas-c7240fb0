## What are Agent Skills?

**Agent Skills** are a standardized way to package capabilities, instructions, and best practices for AI agents. Instead of pasting the same prompt repeatedly ("How do I implement MVVM?", "Check this for accessibility"), you install these skills into your agent's environment.

When an agent detects you are working on a relevant task (e.g., "Create a verified repository"), it automatically loads the expert instructions from the corresponding `SKILL.md` file.

## How It Works

When you ask your AI assistant to help with Android development tasks, it automatically detects which skills are relevant and loads the expert instructions:

```text
You: "Create a repository for fetching news with offline support"

Agent: *Loads android-data-layer skill*
        *Applies Repository Pattern with Room + Retrofit*
        *Implements offline-first sync logic*
```
